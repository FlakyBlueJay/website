# Website

Source code for my website, accessible at https://flaky.blue. Astro-erudite is used as the template.

## Differences from base astro-erudite
- About page removed since I use the index page for that, adapted into projects.
- Section for paused/archived projects, projects can be marked as such with `isArchived`.
- Following were removed as I feel it's unnecessary for me personally:
    - Logo, I don't use one.
    - Authors, feels unnecessary for a personal site only I manage.
- Blog and project cards have rounded corners for the images to keep it consistent with theming
